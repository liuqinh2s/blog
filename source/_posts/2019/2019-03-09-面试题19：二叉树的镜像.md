## 题目

请完成一个函数，输入一个二叉树，该函数输出它的镜像。

二叉树的结点定义如下：

```c++
struct BinaryTreeNode
{
	int m_nValue;
	BinaryTreeNode* m_pLeft;
	BinaryTreeNode* m_pRight;
};
```

<!--more-->

## 解答

先翻转根的左右，然后递归对所有结点进行该操作。

```java
public void Mirror(TreeNode root) {
    if(root==null){
        return;
    }
    TreeNode temp = root.left;
    root.left = root.right;
    root.right = temp;
    Mirror(root.left);
    Mirror(root.right);
}
```
